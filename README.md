## How use?

For developers and programming enthusiasts who want to try it out, do:

1. Clone this repo in your machine.
1. Make sure you have `nodejs` installed on your machine.

- In root dir, execute:
```bash
npm install
```
- After installing all project dependencies, run:
```bash
npm start
```
